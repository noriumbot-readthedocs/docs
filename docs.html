Aliases and Syntaxes

A syntax is the correct way you should use the bot,
e.g. .ban @Noob101 reason. Reasons are always optional, you cannot use IDs to ban members.
An alias is another way to execute a command: e.g. instead of .ban, you do .b. Commands that have a short name do not have an alias.

Permissions

Moderation and Server Management commands cannot be executed by a normal member. They are used to moderate communities.
Custom permissions cannot be set yet: a Dashboard is in progress.

Invalid Syntax

If nothing comes after you execute a command, you have used bad arguments.
Mostly, missing arguments are numbers or @username.

Command Not Found

Simple explanation, the command you executed doesn't exist.

No Permissions

You do not have the permissions to execute a command.
